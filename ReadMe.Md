# 도시철도Set(都市鉄道(韓国)セット)
## 謝辞
きたきゅー様の各種アドオン、M_kasumi~~様~~のITX-青春より改造しました。

主にソウル近辺の都市鉄道(首都圏電鉄)のアドオンです。現役車両を中心に実装しており、ある程度自由度のある組成が出来ます。
## 路線別アドオン解説
